Install OpenCV from Source on macOS

Step 1, you need to have
  1. Xcode and its Xcode Command Line Tools (You might need to login as a member of the Apple Developer Program to download Xcode Command Line Tools. But it seems if you have Xcode fully installed, you will get those command line tools as well, not sure about that.)
  2. Python3 environment. You can use system wide python3 environment, or using the virtual environment (which is the option recommended by me).
  3. Homebrew
Step 2, install pre-requisites.
brew install cmake pkg-config
brew install jpeg libpng libtiff openexr
brew install eigen tbb
Step 3, prepare working directory and clone source codes.
cd ~/Repositories && mkdir opencv && cd opencv
git clone https://github.com/opencv/opencv
git clone https://github.com/opencv/opencv_contrib
cd ~/opencv && mkdir build && cd build
Step 4, configure cmake. Replace OPENCV_EXTRA_MODULES_PATH with your own path.
cmake -D CMAKE_BUILD_TYPE=RELEASE \
    -D CMAKE_INSTALL_PREFIX=/usr/local \
    -D OPENCV_EXTRA_MODULES_PATH=/Users/pharrell/Repositories/opencv/opencv_contrib/modules \
    -D PYTHON3_INCLUDE_DIR=$(python -c "from distutils.sysconfig import get_python_inc; print(get_python_inc())") \
    -D PYTHON_PACKAGES_PATH=$(python3 -c "from distutils.sysconfig import get_python_lib; print(get_python_lib())") \
    -D PYTHON3_EXECUTABLE=$(which python) \
    -D BUILD_opencv_python2=OFF \
    -D BUILD_opencv_python3=ON \
    -D INSTALL_PYTHON_EXAMPLES=ON \
    -D INSTALL_C_EXAMPLES=OFF \
    -D WITH_FFMPEG=ON \
    -D BUILD_EXAMPLES=ON ..
Step 5, build with make -j7 # runs 7 jobs in parallel.
Step 6, install libraries from build dir: sudo make install.
Done.
